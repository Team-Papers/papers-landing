# Papers — Landing Page

Site vitrine de la plateforme **Papers - Livres et Histoires**.
Plateforme camerounaise de publication et lecture de livres numériques avec paiement mobile.

## Stack

- **Next.js 15** (App Router, standalone output)
- **React 19** + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (animations scroll)
- **Lucide React** (icônes)

## Liens

| Environnement | URL |
|---------------|-----|
| **Production** | https://papers237.duckdns.org |

## Développement

```bash
# Installation
npm install

# Serveur de développement
npm run dev

# Build de production
npm run build

# Démarrer le build
npm start
```
